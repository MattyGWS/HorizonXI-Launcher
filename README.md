# HorizonXI Launcher

A native Linux launcher for HorizonXI.

Features

- One-click HorizonXI installation
- Automatic Proton GE installation
- Direct game launch (no official launcher required)
- Dynamic addon management, Ashita plugin and PlayOnline plugin management
- Game settings from within the launcher
- Repair installation
- Nuclear reset (removes all game files like you never hit install, gives you a fresh start)
- Flatpak support

Installation

Download the latest .flatpak from Releases.
